What this does

Ledger takes raw financial statements and produces a deal-ready analysis memo in seconds. Built for deal teams evaluating investment, acquisition, or strategic transaction decisions.

How it works

  1. 01Extract line items from source using the Gemini API
  2. 02Compute 11 financial ratios deterministically in Python
  3. 03Generate deal-analyst commentary grounded in the computed numbers
  4. 04Export the analysis as a Word or PDF memo

Why this design

Large language models hallucinate when doing math. Ledger separates reading from arithmetic: the LLM is used only for extraction and narrative, while all ratio calculations are performed in deterministic Python. The result is analysis that is both fluent and arithmetically correct.
